Kalendar API

/api/ - Es wird alle Dataien angezeigen

Monate:

  • api / months - Alle Monate
  • api / months /[ monate_name ] - z.B. januar (klein geschrieben)
  • api / months /[ monate_name ] / [ veranstaltung_name ] - z.B. holidays, birthdays usw. (klein geschrieben)

Jahre:

  • api / years - Alle Jahre
  • api / months / [ jahr ] - z.B. 2022
